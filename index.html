Center Checklist

Complete the checklist items below and upload the requested photos (max 3 files each, JPG/PNG, max 1MB each).

1) Office room *

One office room per training center with sitting space to accommodate trainers during their office time.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

2) Classroom

The classroom shall have space to accommodate 30 candidates with minimum space of 10 sq.ft./per trainee.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

3) Lab

The Lab shall have space to accommodate 30 candidates with minimum space of 10 sq.ft./per trainee.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

4) Differently abled friendly training center

The training center preferably have ramps, lifts and toilets for differently abled people.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

5) Placement and Entrepreneurship cell

A separate counselling/placement desk shall be available in the training center. An entrepreneurship cell shall be available in the training center.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

6) Corridor

No entry to or exit from academic area should be through another area.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

7) Library

Training center shall have library (physical & digital) facilities for the benefit of trainees.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

8) Toilets for male

Toilets for males and females should be segregated with the appropriate partition. Toilets shall also be disabled-friendly.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

9) Toilets for female

Toilets for males and females should be segregated with the appropriate partition. Toilets shall also be disabled-friendly.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

10) Distance from nearest bus stop, railway station (in meters)

It is advisable to be well connected to different transport locations. It is advisable that a dropping facility for persons with disabilities shall be provided from the transport locations.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

11) Approach road to the center

Safe and secured; the center should be located such that ordinary citizens use the approach roads regularly. It should be away from areas where anti-social elements move about.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

12) Securing of wires

In case of open wiring, all wires should be properly taped and secured with a PVC pipe on the top. It is advisable to have concealed wiring. The building should have 'Electricity safety Certificate' from competent authority.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

13) Biometric device

Aadhar-Enabled Biometric Attendance System (AEBAS) shall be used as defined by Department of Telecommunications / GoI. Or the standard RFID/biometric system used in Andhra Pradesh may be used.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

14) Safe drinking water

Standard: Any one of the following: 1) RO/ozonized treated water. 2) Packaged drinking water certified by ISI or BIS.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

15) First aid kit

First aid kit shall have basic medication like antiseptic soap & lotion, band-aid, ice pouch, tissue pack, etc.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

16) Fire extinguishers

The training center shall have at least two fire extinguishers placed at easily accessible locations (e.g., corridors). The center should have valid 'Fire NOC' or certification from the local fire department.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

17) Protection of stairs, balconies and other locations

All areas should be protected with railings or walls of at least 3 ft. height. If any area is not protected the center should not be commissioned.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

18) Contact details of important people

The emergency contact details shall be displayed at the training center, which shall include: 1) center in-charge, 2) nearest police station, 3) fire station, and other required contacts.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

19) Electric power backup

Alternative power sources like generators, inverters, UPS or solar backups are allowed. Backup should support essential items: CCTV, biometric devices, office computers, domain lab equipment, etc.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

20) Internet Connection

The center should have at least two internet connections. If networked, the primary connection should be at least 20 Mbps for 10 computers on LAN. Non-networked computers should have 10 Mbps each.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

21) CCTV - Specification, location, recording, and storage

CCTV systems should cover important academic and administrative areas. Recording and storage must be available at the training center per prescribed retention rules.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

1) Area (in sq.m)

Area per student in all academic rooms is 10 Sq.ft. Combining of theory and domain lab is permitted only when both theory and practical classes are not taken at the same time for different batches. Seating arrangement must allow all students to see the trainer with an unobstructed view of the writing board and smart TV.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

2) Tables/Chairs seating capacity (no of students)

Chair – one per candidate with writing pad. Tables and benches must have proper space, depth, and height based on average candidate size.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

3) Fans (in number)

1 ceiling fan per each 150 Sq.ft or part thereof. Room size 450 Sq.ft but only 300 Sq.ft used → only 2 fans. Wall-mounted fans = double the ceiling fans. For AC rooms: 2-ton AC for 400 Sq.ft + 0.5 ton per 5 additional people.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

4) Lights

1 fluorescent light (2,400 lumens) for each 150 Sq.ft or part thereof.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

5) Lab (if applicable)

Lab equipment must comply with specifications defined by the respective Sector Skill Council, NCVET, or authorized body.

File Type Allowed: JPG, JPEG, PNG | Max File Size: 1 MB | Max Number of Files: 3

6) Trainers (if applicable)

Trainer qualifications, experience, and certifications must meet NCVET / AB or approved competent authority standards.
